# Project Juniper

![Banner](https://i.imgur.com/p0P4tcI.png)

Project Juniper is a Fortnite private server game server, forked and rebuilt from Project Reboot 3.0 with custom modifications and backend integration.

## Features

1. **S3-S15 Support**: Full support for Fortnite Chapter 1 Season 3 through Chapter 2 Season 5.
2. **S1-S2 & S16-S19 Compatibility**: Seasons 1-2 and 16-19 may not work perfectly.
3. **S20+ Branch**: An experimental branch for S20+ is available but currently unstable.
4. **Custom Backend**: Built to work alongside Juniper Backend for account management, matchmaking, and profiles.
5. **Ongoing Improvements**: Project Juniper is actively being developed with lategame enhancements and bug fixes planned.

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/YOURNAME/Juniper-GameServer.git
   ```
2. Build the project using Visual Studio 2022.
3. Start your Juniper Backend before launching the game server.
4. Run the server using the Juniper Launcher.

## Requirements

- Visual Studio 2022 (C++ Desktop Development workload)
- Juniper Backend running on localhost
- A supported Fortnite build (S3-S15 recommended)
